# BSA15 — REST API

## Описание

Проектирование и документирование REST API: ресурсы, методы HTTP, коды ответов, аутентификация. Разработка OpenAPI/Swagger-спецификации.

## Задачи

- Спроектировать ресурсы REST API
- Описать эндпоинты, методы и форматы данных
- Создать OpenAPI (Swagger) спецификацию

## Артефакты

| Файл | Описание |
|---|---|
| `openapi_spec.yaml` | OpenAPI (Swagger) спецификация |
| `endpoints_description.pdf` | Описание эндпоинтов |
| `request_response_examples.pdf` | Примеры запросов/ответов |
