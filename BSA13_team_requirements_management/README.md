# BSA13 — Управление требованиями (командный проект)

## Описание

Управление жизненным циклом требований: версионирование, трассировка, контроль изменений. Командная работа по поддержке актуальности требований на всём протяжении проекта.

## Задачи

- Выстроить процесс управления изменениями требований
- Создать матрицу трассировки
- Определить базовую линию (baseline) требований

## Артефакты

| Файл | Описание |
|---|---|
| `traceability_matrix.pdf` | Матрица трассировки |
| `change_register.pdf` | Реестр изменений |
| `requirements_baseline.pdf` | Базовая линия требований |
