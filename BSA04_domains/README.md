# BSA04 — Доменный анализ

## Описание

Анализ предметной области: выявление ключевых понятий, построение доменной модели и глоссария. Изучение бизнес-контекста системы.

## Задачи

- Выявить ключевые сущности предметной области
- Составить глоссарий домена
- Построить контекстную диаграмму
