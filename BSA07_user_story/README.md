# BSA07 — User Story (Пользовательские истории)

## Описание

Написание пользовательских историй с критериями приёмки. Декомпозиция эпиков на истории, составление Product Backlog в формате Agile.

## Задачи

- Декомпозировать эпики на User Stories
- Написать User Stories по формату: «Как [роль], я хочу [действие], чтобы [выгода]»
- Определить критерии приёмки (Acceptance Criteria)

## Артефакты

| Файл | Описание |
|---|---|
| `user_stories.pdf` | Набор User Stories |
| `acceptance_criteria.pdf` | Критерии приёмки |
| `product_backlog.pdf` | Product Backlog |

## Навыки

`User Story` `Acceptance Criteria` `Agile` `Product Backlog` `Декомпозиция`
