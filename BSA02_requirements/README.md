# BSA02 — Требования

## Описание

Основы работы с требованиями: типы требований, методы сбора, структурирование и документирование. Базовые техники элицитации.

## Задачи

- Изучить виды требований (бизнес, пользовательские, системные)
- Освоить техники элицитации (интервью, опросы, наблюдение)
- Задокументировать требования в структурированном виде
