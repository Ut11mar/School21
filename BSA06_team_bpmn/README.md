# BSA06 — BPMN (командный проект)

## Описание

Моделирование бизнес-процессов в нотации BPMN 2.0. Описание текущих процессов (AS-IS) и проектирование улучшенных процессов (TO-BE). Командный проект.

## Задачи

- Описать текущие бизнес-процессы (AS-IS)
- Выявить узкие места и возможности для улучшения
- Спроектировать улучшенные процессы (TO-BE)

## Артефакты

| Файл | Описание |
|---|---|
| `process_AS-IS.bpmn` | BPMN-диаграмма AS-IS |
| `process_TO-BE.bpmn` | BPMN-диаграмма TO-BE |
