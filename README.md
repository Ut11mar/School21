# Портфолио: Бизнес- и системный анализ

Учебные проекты по курсу **BSA (Business & Systems Analysis)** в рамках Школы 21.

---

## О курсе

Курс охватывает полный цикл работы бизнес- и системного аналитика: от изучения роли аналитика в команде до интеграции систем и проектирования API.
