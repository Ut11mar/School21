# BSA12 — Нефункциональные требования

## Описание

Определение и документирование нефункциональных требований (NFR): производительность, безопасность, масштабируемость, доступность, удобство использования.

## Задачи

- Определить атрибуты качества системы
- Описать NFR в измеримых показателях
- Приоритизировать требования по ISO 25010

## Артефакты

| Файл | Описание |
|---|---|
| `NFR_document.pdf` | Раздел NFR в SRS |
| `quality_attributes.pdf` | Атрибуты качества (ISO 25010) |
| `NFR_prioritization.pdf` | Приоритизация NFR |
