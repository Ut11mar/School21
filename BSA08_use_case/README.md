# BSA08 — Use Case (Варианты использования)

## Описание

Разработка вариантов использования: Use Case Diagram и детальные описания сценариев. Основной и альтернативные потоки, предусловия и постусловия.

## Задачи

- Построить Use Case Diagram системы
- Написать детальные спецификации для ключевых вариантов использования
- Описать основной и альтернативные потоки

## Артефакты

| `use_case_specifications.pdf` | Спецификации Use Cases |
