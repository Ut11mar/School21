# BSA10 — Функциональные требования

## Описание

Разработка функциональных требований в формате SRS (Software Requirements Specification). Описание функций системы с учётом бизнес-правил и ограничений.

## Задачи

- Описать функции системы в формате SRS
- Определить бизнес-правила
- Расставить приоритеты требований (MoSCoW)

## Артефакты

| Файл | Описание |
|---|---|
| `SRS_document.pdf` | Software Requirements Specification |
| `functional_requirements.pdf` | Функциональные требования |
| `business_rules.pdf` | Бизнес-правила |

## Навыки
