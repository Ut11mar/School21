# BSA01 — Заинтересованные стороны

## Описание

Анализ стейкхолдеров проекта: выявление, классификация и управление заинтересованными сторонами. Построение матрицы интересов и влияния.

## Задачи

- Выявить всех стейкхолдеров проекта
- Классифицировать их по уровню интереса и влияния
- Составить план взаимодействия

## Артефакты

| Файл | Описание |
|---|---|
| `stakeholders_register.pdf` | Реестр стейкхолдеров |
| `interest_influence_matrix.pdf` | Матрица интересов/влияния |
| `communication_plan.pdf` | План взаимодействия |
