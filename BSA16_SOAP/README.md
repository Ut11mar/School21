# BSA16 — SOAP

## Описание

Разработка SOAP-сервиса: WSDL-описание, структура сообщений, описание операций. Сравнение SOAP и REST, понимание сценариев применения.

## Задачи

- Разработать WSDL-описание сервиса
- Определить структуру SOAP-сообщений (запрос/ответ)
- Описать операции сервиса

## Артефакты

| Файл | Описание |
|---|---|
| `service.wsdl` | WSDL-описание сервиса |
| `xsd_schemas.xsd` | XSD-схемы данных |
| `soap_messages_examples.pdf` | Примеры SOAP-сообщений |

## Навыки
