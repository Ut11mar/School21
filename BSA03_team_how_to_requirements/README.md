# BSA03 — Командная работа с требованиями

## Описание

Практика командного сбора и согласования требований. Проведение воркшопов, разрешение конфликтов требований, работа с несогласованными требованиями.

## Задачи

- Организовать и провести воркшоп по требованиям
- Выявить и разрешить конфликты требований
- Согласовать требования со всеми стейкхолдерами

## Артефакты

| Файл | Описание |
|---|---|
| `workshop_protocol.pdf` | Протокол воркшопа |
| `requirements_agreed.pdf` | Согласованные требования |
| `traceability_matrix.pdf` | Матрица трассировки |
