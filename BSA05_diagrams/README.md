# BSA05 — Диаграммы

## Описание

Построение UML-диаграмм для описания структуры и поведения системы: диаграммы классов, последовательности, состояний, компонентов.

## Задачи

- Построить диаграмму классов для предметной области
- Создать диаграмму последовательности для ключевых сценариев
- Разработать диаграмму состояний для основных объектов

## Артефакты

| Файл | Описание |
|---|---|
| `class_diagram.pdf` | Диаграмма классов |
| `sequence_diagram.pdf` | Диаграмма последовательности |
| `state_diagram.pdf` | Диаграмма состояний |
| `component_diagram.pdf` | Диаграмма компонентов |

## Навыки

`UML` `Диаграмма классов` `Диаграмма последовательности` `Диаграмма состояний`
