# BSA09 — БД и данные (командный проект)

## Описание

Проектирование базы данных: концептуальная, логическая и физическая модели данных. Командная работа над структурой хранения данных системы.

## Задачи

- Построить концептуальную модель данных (ER-диаграмма)
- Разработать логическую и физическую модели БД
- Описать сущности, атрибуты и связи

## Артефакты

| Файл | Описание |
|---|---|
| `ER_diagram.pdf` | ER-диаграмма (концептуальная модель) |
| `logical_model.pdf` | Логическая модель данных |
| `ddl_scripts.sql` | DDL-скрипты (физическая модель) |
| `entities_description.pdf` | Описание сущностей |

## Навыки
