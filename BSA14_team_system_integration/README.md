# BSA14 — Системная интеграция (командный проект)

## Описание

Проектирование интеграции систем: интеграционные паттерны, описание точек интеграции, форматы обмена данными. Командная работа над архитектурой взаимодействия систем.

## Задачи

- Определить точки интеграции между системами
- Выбрать и обосновать паттерны интеграции
- Описать форматы обмена данными и контракты API
