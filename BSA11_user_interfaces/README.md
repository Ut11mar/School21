# BSA11 — Пользовательские интерфейсы

## Описание

Проектирование пользовательских интерфейсов: wireframes, прототипы, описание UI/UX-требований. Анализ пользовательских сценариев взаимодействия с системой.

## Задачи

- Создать wireframes ключевых экранов
- Описать UX-требования и пользовательские сценарии
- Разработать прототип интерфейса

## Артефакты

| Файл | Описание |
|---|---|
| `wireframes.pdf` | Wireframes экранов |
| `screen_prototypes.pdf` | Прототипы экранов |
| `ux_requirements.pdf` | Требования к UI/UX |

## Навыки

`UI/UX` `Wireframe` `Прототипирование` `Пользовательские сценарии`
